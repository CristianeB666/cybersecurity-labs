# Cybersecurity Projects | Projetos de Cibersegurança

This repository contains projects, scripts, and detection rules developed for cybersecurity studies and practical security monitoring environments.

The main goal is to share knowledge, improve defensive capabilities, and document experiments related to Security Operations (SOC), threat detection, and security automation.

- Wazuh detection rules
- Security monitoring
- Log analysis
- Python scripts for cybersecurity
- SOC laboratory experiments
- Threat detection and incident analysis

This repository serves as a personal knowledge base and portfolio of cybersecurity experiments and implementations.
